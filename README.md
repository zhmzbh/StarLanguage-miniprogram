# 星语天文 · 微信小程序

> 为天文爱好者打造的科普工具，包含天象预报、星空简图、天文百科和订阅提醒。

## 功能
- 🌠 今日天象：展示近期天文事件，支持订阅提醒
- ✨ 星空简图：Canvas 绘制北斗七星，支持拖拽平移
- 📚 天文百科：云数据库存储科普文章
- 🔔 消息推送：微信订阅消息提醒（需模板）

## 技术栈
- 微信小程序原生框架 + 云开发
- Canvas 2D 绘图（兼容旧版）
- 云函数 + 云数据库

## 如何运行
1. 克隆本仓库
2. 使用微信开发者工具导入，填入你自己的 AppID
3. 开通云开发，创建集合 `knowledge` 并插入数据
4. 部署所有云函数（getKnowledge, sendSubscribe）
5. 修改 `index.js` 中的订阅消息模板 ID
6. 编译运行
